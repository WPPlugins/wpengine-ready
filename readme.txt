== Description ==

This plugin analyzes your WordPress installation and assesses whether it is ready for migration to WPEngine hosting. If it is not ready it makes recommendations. 

== Installation ==

1. Upload `wpengine-ready` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
